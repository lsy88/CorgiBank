## ✨ 主要功能

*   **👥 员工管理**：轻松添加、修改和管理员工信息。
*   **📦 产品与原料管理**：
    *   支持产品和原料的增删改查。
    *   灵活设置项目的单价/成本。
*   **🏭 生产批次管理**：
    *   **批次化录入**：按生产批次管理数据，关联该批次下的产出产品、消耗原料。
    *   **快照定价**：自动保存批次创建时的单价信息，确保历史数据的准确性。
    *   **灵活分配**：支持多名员工参与同一批次，并可设置不同的工资分配比例。
*   **📊 工资报表**：
    *   **多维度统计**：支持“按员工”或“按批次”两种视图查看月度/年度数据。
    *   **成本核算**：自动计算总产值（工资支出）与总原料成本。
    *   **Excel 导出**：支持一键导出详细报表，包含批次概览、生产详情、原料详情、员工薪资详情四个独立工作表，数据更清晰。
*   **💾 数据安全**：
    *   数据本地持久化存储（JSON格式），无需联网，安全可靠。
    *   支持一键打开数据文件夹进行备份。
*   **🎨 现代化界面**：
    *   基于 Ant Design 的清爽 UI。
    *   大窗口设计 (1200x900)，数据展示更清晰。
    *   左侧菜单栏支持折叠。

## 🛠️ 技术栈

*   **Core**: Electron
*   **Frontend**: React, TypeScript, Vite
*   **UI Framework**: Ant Design
*   **Data Persistence**: Node.js fs (Local JSON)

## 🚀 快速开始

### 开发环境

1.  **安装依赖**
    ```bash
    npm install
    ```

2.  **启动开发模式**
    ```bash
    npm run dev
    ```

### 打包构建

构建 Windows 安装包：

```bash
npm run build
```

构建完成后，安装包位于 `release` 目录下。

## 📂 数据存储

应用数据存储在系统的用户数据目录下。
*   **查看方式**：点击应用左下角的 <img src="https://api.iconify.design/ant-design:folder-open-outlined.svg" width="16" style="vertical-align: middle;" /> **数据位置** 按钮即可直接打开数据文件夹。
*   **备份建议**：定期备份该文件夹下的 `corgibank-data.json` 文件。
